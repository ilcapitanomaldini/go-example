# README

Contains the following APIs:
    - Login
    - Logout
    - Change Password
    - Forgot Password

### Usage

To run, hit `go run .` and go to localhost:9000 to view the pages. Or alternatively, interact with the API with an API browser/helper.
To test, hit `go test`.
